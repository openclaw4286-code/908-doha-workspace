# 908 Doha · Mobile UI Kit

Reference build that uses the 908 Design System end-to-end.

## Files
- `index.html` — interactive prototype. Navigate Home → Send → Confirm → Result; toggle light/dark.
- `components.jsx` — primitives: `Phone`, `Top`, `Button`, `ListRow`, `Badge`, `Switch`, `Icon`.
- `screens.jsx` — screen compositions: `HomeScreen`, `SendScreen`, `ResultScreen`, `SettingsScreen`, `AddCardSheet`.

## Run
Open `index.html`. No build step.

## Coverage
Home (balance, quick actions, insight chart, activity rows), Send (amount + quick-add chips + validation), Result (success with Fraunces headline), Settings (dark-mode switch + list sections), Add Card bottom sheet, Toast, Bottom tab bar.
